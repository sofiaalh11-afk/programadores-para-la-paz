Sistema operativo:Es el software principal que administra los recursos del computador o servidor. Permite ejecutar aplicaciones y gestionar procesos.
Node.js:Es un entorno que permite ejecutar código JavaScript en el servidor. Se utiliza para crear aplicaciones y servicios web.
Aplicación server.js:Es el archivo que contiene la lógica del servidor. Define las rutas y respuestas que recibirán los usuarios.
Puerto:Es un punto de comunicación por donde el servidor recibe solicitudes. En esta práctica se utiliza el puerto 3000.
Logs:Son registros que almacenan eventos y actividades del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
