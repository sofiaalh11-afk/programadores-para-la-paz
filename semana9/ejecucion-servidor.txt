Al ejecutar el comando node server.js, el servidor se inicia y queda disponible para recibir solicitudes desde el navegador.
En la terminal aparece el mensaje: "Servidor ejecutándose en puerto 3000".
Cuando se visita la ruta /estado, el servidor responde con información en formato JSON indicando que el servicio está funcionando correctamente y que corresponde a una API comunitaria.
Respuesta obtenida:
{
"estado": "Servidor funcionando",
"servicio": "API comunitaria"
}
